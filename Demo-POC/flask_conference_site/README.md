# TechConf 2025 Website

A Flask-based website for a 1-day technical conference.

## Features
- **Schedule View**: Browse all sessions with time, title, speaker, and category.
- **Search**: Filter sessions by title, speaker, or category.
- **Speaker Details**: View speaker names and links to their LinkedIn profiles.
- **Responsive Design**: Works on desktop and mobile.

## Setup

1.  **Prerequisites**: Python 3.x installed.
2.  **Install Dependencies**:
    ```bash
    pip install -r requirements.txt
    ```
3.  **Run the Application**:
    ```bash
    python app.py
    ```
4.  **Access the Website**:
    Open your browser and navigate to `http://127.0.0.1:5000`.

## Project Structure
- `app.py`: Main Flask application.
- `data.py`: Dummy data for sessions.
- `templates/`: HTML templates.
- `static/`: CSS styles.
